## 1. Como leer esta propuesta

Este documento no busca cerrar el portafolio final. Busca poner una version conversable sobre la mesa para que podamos revisar:

1. si los servicios tienen sentido,
2. si estan bien diferenciados,
3. si el precio sugerido corresponde al valor,
4. si conviene cobrarlos como proyecto o como mensualidad,
5. cuanto tiempo toma implementarlos,
6. cuanto tiempo conviene que una empresa permanezca en cada escalon antes de avanzar.

La idea central es que VectorIA no vende piezas sueltas ni solo diseno. Vende avances comerciales concretos segun el momento real de cada empresa.

Para esta V1, la lectura comercial debe ser mas agresiva de entrada: precios y tiempos mas bajos, solo el producto 1 como pago unico real y una ruta anual para clientes que entren a continuidad.

## 2. Logica general del portafolio

La propuesta esta pensada como una escalera de madurez comercial. No todas las empresas entran por el mismo punto, pero cada servicio responde a una necesidad clara.

Uno de los grandes diferenciales de VectorIA es su capacidad de trabajar SEM de forma transversal en todo el portafolio.

Pero SEM no significa lo mismo en todos los escalones.

La funcion correcta de SEM cambia segun el momento del cliente:

1. en escalones tempranos sirve para sondeo y validacion,
2. en escalones intermedios sirve para probar claridad y conversion,
3. en captacion sirve como motor estructurado de demanda,
4. en continuidad sirve como soporte, refuerzo o amplificacion.

No debe leerse como una secuencia obligatoria para todos los clientes.

Un negocio nuevo normalmente entra por Lanzamiento Comercial Inicial.

Una empresa que ya tiene algo montado, pero necesita ordenar, mejorar conversion o probar una relacion de menor riesgo, puede entrar directo a Presencia que Vende.

| Escalon | Servicio | Que resuelve | Tipo de cobro sugerido |
| :-- | :-- | :-- | :-- |
| 1 | Lanzamiento Comercial Inicial | salir al mercado desde cero con base minima util | Un solo pago |
| 2 | Presencia que Vende | reorganizar y optimizar una presencia ya existente para convertir mejor | Plan mensual con compromiso anual |
| 3 | Funnel de Captacion | generar demanda medible y aprendizaje comercial | Plan mensual con compromiso anual |
| 4 | Motor de Contenido Funcional | sostener presencia, confianza y apoyo comercial en el tiempo | Plan mensual con compromiso anual |

### Paso transversal obligatorio

**Impulso Inicial de Demanda:** activacion breve de SEM o pauta inicial para validar traccion sobre una presencia ya clara. No es todavia Funnel de Captacion completo, pero si es el paso obligatorio de sondeo para decidir si el cliente ya puede pasar a ese nivel.

### Complementarios basicos opcionales

Ademas de los modulos principales, la propuesta comercial necesita una capa chica de complementarios basicos opcionales.

Esto sirve para no dejar huecos cuando el cliente necesita infraestructura minima para poder ejecutar bien el servicio principal.

La lectura correcta es esta:

1. no son productos centrales del portafolio,
2. no deben robar protagonismo a la escalera,
3. no se incluyen automaticamente,
4. se activan cuando habilitan mejor la entrega principal.

Los complementarios basicos sugeridos para esta V1 son:

| Complementario | Que resuelve | Tipo de cobro sugerido |
| :-- | :-- | :-- |
| Dominio y DNS basico | darle identidad digital minima al activo principal | Configuracion puntual mas costo del proveedor |
| Hosting basico | dejar operando landing, pagina simple o activo principal | Configuracion puntual mas costo del proveedor |
| Correo profesional basico | profesionalizar comunicacion comercial y operativa | Configuracion puntual mas licencias o costo del proveedor |

La regla comercial correcta es separar siempre:

1. el trabajo de gestion o configuracion de la agencia,
2. el costo directo del proveedor,
3. la renovacion o licencia recurrente cuando exista.

## 3. Servicios propuestos

### 3.1 Lanzamiento Comercial Inicial

**Que es:**

Un servicio de salida al mercado para empresas o unidades de negocio que todavia estan en cero o casi en cero y necesitan una base comercial digital minima para empezar a vender.

**Que justifica este servicio:**

1. evita que el cliente arranque improvisando,
2. concentra los activos minimos clave en un esfuerzo corto,
3. reduce el tiempo para empezar a vender o validar,
4. deja una base que luego permite ordenar mejor captacion, contenido o seguimiento.

**Cuando tiene sentido venderlo:**

1. cuando la empresa apenas va a lanzar una oferta,
2. cuando aun no existe una presencia digital minima util,
3. cuando no hay aun una estructura minima de contacto o conversion,
4. cuando el cliente necesita velocidad mas que sofisticacion.

**Diferencia clave frente al producto 2:**

Aqui no se trata de ordenar una presencia ya existente. Aqui se trata de construir la primera base comercial funcional para salir al mercado.

En negocios locales, esa base minima puede incluir desde el inicio la activacion de Google Business como punto de visibilidad, confianza y contacto.

**Funcion de SEM en este escalon:**

Aqui SEM no se vende como performance completo. Se usa como herramienta inicial de sondeo para validar si la oferta despierta interes, si la ruta de contacto responde y si el cliente ya puede recibir primeras señales de demanda.

**Lectura comercial recomendada para V1:** este producto conviene cerrarlo como paquete fijo de entrada, no como rango abierto.  
**Precio sugerido:** 12,000 MXN  
**Modalidad sugerida:** un solo pago  
**Tiempo de implementacion:** 5 a 10 dias habiles  
**Tiempo recomendado en este escalon:** 1 a 2 meses

**Incluye exactamente:**

1. una landing o pagina de arranque de una sola seccion comercial,
2. propuesta de valor principal redactada,
3. copy base de presentacion comercial,
4. una sola ruta principal de contacto por WhatsApp o formulario,
5. configuracion basica de identidad visual funcional aplicada al activo,
6. tres piezas graficas base de lanzamiento,
7. texto base de descripcion comercial para perfil o pagina,
8. checklist final de salida a mercado.

**Variable permitida dentro del mismo precio:**

1. Google Business Profile si el negocio es local,
2. o un solo perfil principal como Facebook, Instagram o LinkedIn,
3. pero no todos al mismo tiempo.

**No incluye dentro del precio fijo:**

1. sitio de multiples secciones,
2. branding profundo,
3. ecommerce,
4. dominio, hosting o correos,
5. campañas mensuales,
6. community management,
7. multiples perfiles a la vez,
8. cambios ilimitados.

**Cuanto tiempo deberia quedarse una empresa aqui:**

Este escalon no deberia volverse permanente. Sirve para arrancar y validar base. Si despues de 1 a 2 meses la empresa sigue aqui, normalmente significa que aun no termina de aclarar su oferta, su proceso o su foco comercial.

La utilidad comercial de dejarlo fijo es alta: se vende mas facil, baja friccion, captura antes al cliente y permite abrir despues continuidad dentro del ecosistema.

### 3.2 Presencia que Vende

**Que es:**

Un servicio para reorganizar y optimizar la presencia digital de un negocio que ya salio al mercado, pero cuya cara comercial sigue siendo confusa, debil o poco convincente.

**Que justifica este servicio:**

1. muchas empresas ya venden algo bueno, pero se presentan de forma confusa,
2. una presencia mejor ordenada puede mejorar conversion sin rehacer todo el negocio,
3. suele ser una intervencion clara, visible y facil de entender comercialmente,
4. prepara el terreno para invertir mejor en trafico o en contenido.

**Cuando tiene sentido venderlo:**

1. cuando el cliente ya existe pero su sitio, landing o mensaje no convierten bien,
2. cuando hay desorden entre oferta, narrativa y llamados a la accion,
3. cuando todavia no conviene meter pauta porque primero hay que ordenar la base,
4. cuando el frente comercial digital no representa bien el valor real del negocio.
5. cuando el cliente quiere empezar con una intervencion mas acotada por presupuesto o para probarnos antes de escalar.

**Diferencia clave frente al producto 1:**

Aqui no estamos sacando a la empresa de cero. Estamos tomando una presencia ya existente y la estamos volviendo mas clara, mas confiable y mas convertible.

Si Google Business ya existe, aqui si puede entrar su reorganizacion u optimizacion como parte del ordenamiento de presencia.

**Funcion de SEM en este escalon:**

Aqui SEM sirve para probar si la presencia ya ordenada convierte mejor. Su valor no esta en escalar presupuesto todavia, sino en medir claridad comercial, calidad de respuesta y readiness para el siguiente paso.

**Precio sugerido:** 6,000 a 10,000 MXN al mes  
**Modalidad sugerida:** plan mensual con compromiso anual  
**Tiempo de implementacion:** 7 a 15 dias habiles de arranque  
**Tiempo recomendado en este escalon:** 3 a 4 meses como modulo dominante dentro de una ruta anual

**Cuanto tiempo deberia quedarse una empresa aqui:**

No conviene leer este modulo como relacion corta aislada. La idea correcta es que opere entre 3 y 4 meses como modulo dominante dentro de una relacion anual con el cliente.

### 3.2.1 Impulso Inicial de Demanda

**Que es:**

Un paso obligatorio de activacion breve de SEM o pauta digital para validar si una presencia ya clara puede atraer interes real y si el cliente esta listo para pasar a captacion formal.

**Que justifica este subcomponente:**

1. permite probar traccion antes de vender un funnel completo,
2. ayuda a validar readiness comercial y capacidad de seguimiento,
3. da una lectura inicial de respuesta del mercado,
4. funciona como criterio de paso entre presencia y captacion.

**Cuando tiene sentido venderlo:**

1. cuando la base ya esta suficientemente clara,
2. cuando existe una pagina o activo de entrada funcional,
3. antes de formalizar captacion,
4. cuando se necesita validar interes real y capacidad de seguimiento.

**Que puede incluir:**

1. configuracion basica de una campana inicial,
2. copy base de anuncio,
3. una variante creativa funcional,
4. trafico hacia pagina o activo optimizado,
5. lectura inicial de respuesta y calidad del contacto.

**Que no incluye:**

1. gestion continua de performance,
2. optimizacion intensiva semanal,
3. multiples embudos activos,
4. escalamiento de presupuesto como servicio completo,
5. sustitucion del Funnel de Captacion.

**Que determina:**

1. si la oferta esta lista para recibir mas trafico,
2. si la respuesta del mercado justifica avanzar,
3. si el cliente tiene capacidad operativa para atender contactos,
4. si el siguiente paso correcto si es Funnel de Captacion.

**Funcion de SEM en este paso:**

Aqui SEM funciona como metodo formal de sondeo. No busca volumen por si mismo. Busca evidencia para decidir si el cliente realmente esta preparado para pasar a captacion estructurada.

**Precio sugerido:** 3,000 a 6,000 MXN mas inversion publicitaria  
**Modalidad sugerida:** un solo pago por activacion  
**Tiempo de implementacion:** 3 a 5 dias habiles  
**Tiempo recomendado en este escalon:** 2 a 3 semanas de lectura inicial

### 3.3 Funnel de Captacion

**Que es:**

Un servicio para activar demanda y convertir trafico en prospectos utiles, con una logica mas cercana a resultado economico que a presencia visual.

**Que justifica este servicio:**

1. conecta el trabajo de marketing con una lectura mas directa de negocio,
2. permite aprender que mensaje, oferta o canal responde mejor,
3. ayuda a medir si la empresa realmente esta lista para escalar captacion,
4. puede convertirse en la puerta de entrada a relaciones de mayor ticket o continuidad.

**Cuando tiene sentido venderlo:**

1. cuando ya existe una oferta clara,
2. cuando la empresa ya tiene una base minima de conversion,
3. cuando vale la pena poner trafico frente a un sistema que ya puede responder,
4. cuando el cliente quiere demanda, no solo orden visual.

**Funcion de SEM en este escalon:**

Aqui SEM ya es el motor principal del servicio. Su papel es atraer trafico de intencion, convertirlo en contactos utiles y generar una lectura medible para optimizar y escalar.

**Precio sugerido:** 8,000 a 15,000 MXN al mes mas inversion publicitaria  
**Modalidad sugerida:** plan mensual con compromiso anual  
**Tiempo de implementacion:** 5 a 10 dias habiles de activacion inicial  
**Tiempo recomendado en este escalon:** 3 a 4 meses como modulo dominante dentro de una ruta anual

**Cuanto tiempo deberia quedarse una empresa aqui:**

No conviene leer este modulo como una prueba muy corta. La idea correcta es trabajar entre 3 y 4 meses como modulo dominante dentro de una relacion anual, para aprender, ajustar y despues sostener o escalar.

### 3.4 Motor de Contenido Funcional

**Que es:**

Un servicio recurrente para producir contenido util que sostenga presencia, confianza y apoyo comercial sin depender de produccion artesanal lenta.

**Que justifica este servicio:**

1. muchas empresas no fallan por falta de ideas sino por falta de continuidad,
2. el contenido funcional ayuda a explicar, demostrar criterio y reforzar conversion,
3. puede alimentar presencia, remarketing, seguimiento y confianza,
4. es un servicio con sentido si se limita por foco y no por volumen infinito.

**Cuando tiene sentido venderlo:**

1. cuando ya existe una narrativa base,
2. cuando el cliente necesita sostener visibilidad y confianza,
3. cuando el negocio tiene algo que demostrar o educar de forma continua,
4. cuando ya vale la pena trabajar por sistema y no por piezas aisladas.

**Funcion de SEM en este escalon:**

Aqui SEM no necesariamente es la promesa central, pero si puede funcionar como refuerzo tactico: remarketing, apoyo a temporadas, amplificacion de piezas clave o soporte a campañas puntuales.

**Precio sugerido:** 6,000 a 12,000 MXN al mes  
**Modalidad sugerida:** plan mensual con compromiso anual  
**Tiempo de implementacion:** 5 a 10 dias habiles de setup editorial  
**Tiempo recomendado en este escalon:** 6 a 12 meses dentro de una ruta anual

**Cuanto tiempo deberia quedarse una empresa aqui:**

Este escalon es el mejor candidato para la permanencia larga. La lectura correcta es 6 a 12 meses dentro de una ruta anual del cliente y, si funciona, puede quedarse como relacion recurrente sostenida.

## 4. Como se justifica el precio

La logica del precio no esta basada en cobrar horas sueltas. Se sostiene en cinco elementos:

1. claridad del problema que resuelve,
2. impacto comercial potencial,
3. velocidad de implementacion,
4. nivel de criterio y supervision requerido,
5. riesgo de desborde operativo.

## 4.1 Como se cobran los complementarios basicos

Los complementarios basicos no deben venderse como si fueran un producto transformacional.

Se cobran como habilitadores tecnicos del producto principal.

La lectura correcta es esta:

1. si el cliente ya tiene dominio, hosting o correo resueltos, no se le vuelve a cobrar por eso,
2. si no lo tiene, la agencia puede ofrecer resolverlo como complemento,
3. el precio de la agencia cubre configuracion, criterio y puesta en marcha,
4. el costo del proveedor o licencia debe quedar separado.

### Rangos sugeridos V1 para complementarios

| Complementario | Precio sugerido | Modalidad sugerida | Tiempo de implementacion |
| :-- | :-- | :-- | :-- |
| Dominio y DNS basico | 500 a 1,500 MXN mas costo del proveedor | Pago unico por configuracion | 1 a 2 dias habiles |
| Hosting basico | 800 a 2,500 MXN mas costo del proveedor | Pago unico por configuracion | 1 a 3 dias habiles |
| Correo profesional basico | 500 a 1,500 MXN mas licencias o costo del proveedor | Pago unico por configuracion | 1 a 2 dias habiles |

Estos complementarios deben leerse como base tecnica minima, no como upsell inflado.

En otras palabras: no cobramos solo ejecucion. Cobramos foco, sistema, velocidad y capacidad de reducir incertidumbre comercial.

## 5. Como leer el rango de precio

**Tramo bajo del rango:**

1. una sola oferta principal,
2. pocos activos,
3. aprobacion rapida,
4. foco muy claro,
5. baja complejidad operativa.

**Tramo medio del rango:**

1. mas activos por ordenar,
2. mayor coordinacion,
3. uno o dos canales relevantes,
4. ciertos ajustes de mensaje o estructura.

**Tramo alto del rango:**

1. mayor complejidad del negocio,
2. mas trabajo de foco y priorizacion,
3. mayor supervision,
4. mas riesgo comercial si se ejecuta mal,
5. mas esfuerzo sin salir del alcance base.

## 6. Regla clave para no romper alcance

No conviene prometer que el paquete cubre toda la complejidad de una empresa solo porque tiene muchas ofertas, lineas o productos.

La regla deberia ser esta:

**cada servicio trabaja sobre una unidad comercial focal, no sobre todo el universo del cliente.**

Eso significa:

1. si el cliente tiene una sola oferta, el modelo base entra perfecto,
2. si tiene de 2 a 5 ofertas, se elige una principal y una secundaria opcional,
3. si tiene mas de 5 ofertas o un catalogo amplio, se trabaja por categoria, linea o prioridad comercial,
4. si tiene cientos o miles de productos, ya no entra igual y probablemente requiere una variante especifica.

## 7. Recomendacion para conversar con socios

Lo mas util para la revision conjunta no es solo preguntar si los servicios suenan bien. Conviene revisar estas decisiones:

1. si los 4 servicios son suficientes para salir al mercado,
2. si Lanzamiento Comercial Inicial y Presencia que Vende estan suficientemente separados,
3. si Funnel de Captacion se vende solo como setup o tambien como continuidad,
4. si Motor de Contenido Funcional debe ofrecerse desde el inicio o solo despues de cierto readiness,
5. que limites duros debe tener cada servicio para no perder margen.

## 8. Resumen ejecutivo rapido

La propuesta actual si tiene forma de portafolio comercial inicial.

Su fortaleza esta en que cada servicio responde a un momento distinto del cliente:

1. arrancar,
2. ordenar,
3. captar,
4. sostener.

Si la acotamos bien, puede ser una oferta clara, rentable y facil de explicar. Si no la acotamos, el riesgo principal es confundir escalones o meter demasiada complejidad dentro del mismo precio.
