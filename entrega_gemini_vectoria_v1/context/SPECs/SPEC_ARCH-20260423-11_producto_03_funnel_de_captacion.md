## Titulo

Producto 03: Funnel de Captacion

## Estado

Borrador de definicion ejecutiva

## Fecha

2026-04-23

## Proposito del Producto

Funnel de Captacion es el producto diseñado para clientes que ya cuentan con una base comercial minima funcional y ahora necesitan generar demanda de forma mas estructurada, medible y optimizable.

Su funcion no es solamente activar anuncios. Su funcion es construir una capa real de captacion que conecte mensaje, trafico, punto de conversion y lectura inicial de resultados.

## Problema que Resuelve

Este producto resuelve situaciones como:

1. el cliente ya tiene presencia, pero no genera flujo constante de prospectos,
2. existen activos base, pero no hay una estructura clara de captacion,
3. el negocio quiere crecer, pero depende de referencias o trafico organico inestable,
4. ya hubo una prueba inicial de demanda y ahora hace falta formalizar la captacion,
5. el cliente necesita invertir en acquisition de forma mas ordenada y menos improvisada.

## Tipo de Cliente al que Aplica

Aplica principalmente a:

1. negocios con oferta ya definida,
2. clientes con punto de conversion funcional,
3. empresas con capacidad minima de atender leads,
4. clientes que ya pasaron por una etapa de lanzamiento o de orden de presencia,
5. negocios que quieren generar prospectos de forma mas consistente.

## No Aplica Bien a

No aplica bien a:

1. clientes sin oferta clara,
2. negocios sin pagina, landing o canal de conversion funcional,
3. clientes que todavia no pueden atender demanda,
4. proyectos que necesitan primero orden de presencia o lanzamiento,
5. empresas que esperan resultados de performance sin estructura minima de seguimiento.

## Objetivo del Producto

Implementar una estructura de captacion digital suficientemente clara para atraer interes, conducirlo hacia un punto de conversion y generar datos iniciales de rendimiento para optimizar.

## Resultado Esperado

Al terminar este producto, el cliente debe quedar con:

1. una ruta de captacion activada,
2. trafico dirigido hacia un punto de conversion funcional,
3. una estructura inicial de anuncios y mensajes,
4. lectura basica de resultados,
5. evidencia suficiente para decidir si conviene continuar, optimizar o escalar.

## Promesa Correcta del Producto

No prometemos magia, volumen infinito ni resultados garantizados por el simple hecho de prender campañas.

Prometemos construir una capa inicial y seria de captacion para generar aprendizaje real, demanda medible y base de optimizacion.

## Entregables Base

Los entregables de este producto deben enfocarse en estructura, activacion y lectura inicial.

### Entregables propuestos

1. diagnostico de readiness para captacion,
2. definicion de objetivo principal de campaña,
3. estructura basica de funnel o ruta de captacion,
4. configuracion inicial de campaña en Google Ads o Meta Ads segun caso,
5. piezas y copies funcionales para activacion,
6. conexion con landing, formulario o canal de contacto,
7. configuracion basica de medicion inicial,
8. lectura de primeros resultados y recomendaciones siguientes.

## Activos Reales del Producto

Este producto debe trabajar sobre activos concretos de captacion y conversion.

## Activos Base Incluidos

Propuesta inicial de activos base:

1. una ruta de captacion principal,
2. configuracion de una campaña base,
3. set inicial de anuncios o variantes funcionales,
4. copy principal de captacion,
5. conexion a una landing o punto de conversion,
6. CTA y canal de contacto alineados,
7. configuracion basica de tracking o lectura inicial,
8. reporte corto de activacion y observaciones iniciales.

## Activos Opcionales Segun Diagnostico

Los activos opcionales solo deben activarse si ayudan a mejorar el aprendizaje o la conversion sin romper el alcance.

Propuesta inicial de activos opcionales:

1. una segunda variante de anuncio o creatividad,
2. una segunda audiencia o segmentacion controlada,
3. un segundo canal de captacion puntual,
4. ajuste basico de una pagina complementaria,
5. configuracion minima de remarketing si aplica,
6. una segunda CTA o ruta de contacto,
7. adaptaciones creativas adicionales,
8. integracion puntual a un CRM ligero o registro de leads.

## Criterio sobre Activos

Este producto no debe convertirse en una operacion ilimitada de performance desde el primer momento.

Debe enfocarse en una estructura inicial que permita aprender rapido sin dispersar presupuesto ni complejidad.

La seleccion correcta depende de:

1. el canal con mayor probabilidad de respuesta,
2. el nivel de claridad de la oferta,
3. la calidad del punto de conversion,
4. la capacidad del cliente para responder,
5. el objetivo comercial inmediato.

La promesa correcta no es "activamos todo en todos lados".

La promesa correcta es:

**ponemos a trabajar una capa inicial de captacion para generar aprendizaje y demanda medible.**

## Incluye

El producto puede incluir, en su version estandar:

1. sesion breve de diagnostico de readiness,
2. definicion del objetivo de captacion,
3. configuracion de una campaña base,
4. una estructura inicial de anuncios,
5. copies funcionales de captacion,
6. alineacion con landing o punto de conversion,
7. lectura inicial de datos,
8. recomendaciones de siguiente paso.

Cuando aplique segun diagnostico y alcance cerrado, tambien puede incluir variaciones creativas, una segunda segmentacion o una integracion minima a registro de leads.

## No Incluye

Para proteger alcance y rentabilidad, este producto no debe incluir por defecto:

1. gestion ilimitada de performance,
2. escalamiento continuo de campañas sin control,
3. multiples embudos completos al mismo tiempo,
4. automatizaciones avanzadas de marketing,
5. CRM complejo,
6. contenido recurrente permanente,
7. rediseño total de landing o sitio,
8. consultoria comercial extensa,
9. revisiones abiertas sin control,
10. impresion, fabricacion o entrega fisica directa de materiales.

## Regla sobre Activos Fisicos

Este producto pertenece a una agencia de ejecucion digital.

Por lo tanto:

1. activa trafico y conversion en digital,
2. no incluye materiales impresos,
3. no incluye activos tangibles producidos directamente,
4. no contempla logistica fisica como parte del servicio regular.

Si el cliente necesita un apoyo fisico puntual, solo podra manejarse como excepcion:

1. fuera del alcance base,
2. con costo separado,
3. mediante tercero,
4. sin desviar el foco principal del funnel digital.

## Que Si Recibe el Cliente

El cliente recibe una estructura inicial de captacion puesta en marcha con criterio y medicion suficiente para aprender del mercado.

Eso puede traducirse en:

1. una campaña activa,
2. mensajes de captacion funcionales,
3. trafico dirigido,
4. punto de conversion operando,
5. primeros datos de respuesta,
6. evidencia sobre si vale la pena continuar o escalar.

## Que No Debe Esperar Recibir

El cliente no debe interpretar este producto como una gestion total de performance sin limites ni como garantia automatica de ventas.

No debe esperar recibir dentro de este paquete:

1. leads ilimitados,
2. optimizacion continua indefinida,
3. todos los canales activados al mismo tiempo,
4. estrategia omnicanal completa,
5. operacion comercial interna resuelta,
6. cierres de ventas por parte de la agencia,
7. materiales fisicos incluidos,
8. crecimiento escalado sin una fase previa de lectura y ajuste.

La promesa del producto es concreta:

**activar una captacion medible y aprender rapido si el sistema comercial del cliente esta listo para escalar.**

## Prerrequisitos Minimos

Para poder ejecutar este producto, el cliente debe tener al menos:

1. una oferta claramente definible,
2. un punto de conversion funcional,
3. un medio claro de atender contactos,
4. capacidad minima de seguimiento,
5. una persona que apruebe con rapidez,
6. disposicion real para invertir en captacion.

## Flujo de Ejecucion del Producto

El flujo sugerido es:

1. prediagnostico de madurez,
2. validacion de readiness para captacion,
3. definicion del objetivo de campaña,
4. configuracion de activos y anuncios,
5. activacion de trafico hacia punto de conversion,
6. lectura inicial de respuesta,
7. entrega de observaciones y siguiente decision.

## Criterios de Supervision Humana

Como es un producto AI-assisted, la supervision humana debe revisar explicitamente:

1. coherencia entre anuncio y oferta,
2. calidad funcional del mensaje,
3. consistencia entre trafico y punto de conversion,
4. lectura correcta de la respuesta inicial,
5. que no se saquen conclusiones prematuras con datos insuficientes.

## Criterios de Calidad

El producto se considera bien entregado si:

1. la campaña queda activada correctamente,
2. existe una ruta clara desde anuncio hasta contacto,
3. se obtienen primeros datos utilizables,
4. se identifican aprendizajes reales del mercado,
5. el cliente queda en mejor posicion para decidir si continua, corrige o escala.

## Riesgos del Producto

Los principales riesgos a controlar son:

1. vender captacion a clientes que aun no tienen base,
2. activar trafico hacia un punto de conversion debil,
3. confundir prueba de captacion con gestion completa de performance,
4. depender demasiado pronto de multiples variables,
5. que el cliente no tenga capacidad de respuesta y culpe a la pauta.

## Nivel en la Escalera del Cliente

Este producto corresponde a una etapa de crecimiento inicial dentro de la escalera.

Resuelve la necesidad de pasar de base funcional a generacion estructurada de demanda.

## Siguiente Escalon Natural

Si el producto funciona y el cliente demuestra capacidad de respuesta, los siguientes escalones naturales pueden ser:

1. un esquema mas continuo de captacion y optimizacion,
2. Motor de Contenido Funcional, si necesita sostener autoridad y soporte de conversion,
3. Operacion Comercial Ligera, si el cuello de botella principal se vuelve seguimiento y gestion de leads.

## Criterio de Ready for Upsell

El cliente puede subir al siguiente nivel si:

1. ya responde al trafico con un punto de conversion funcional,
2. existe aprendizaje suficiente sobre mensaje y canal,
3. puede atender contactos con rapidez,
4. el problema deja de ser activacion y pasa a ser optimizacion o continuidad,
5. ya tiene sentido una relacion mas recurrente de captacion o soporte comercial.

## Valor Estrategico para la Agencia

Este producto sirve como puente entre base comercial y crecimiento.

Su valor estrategico esta en que:

1. conecta la agencia con resultado mas cercano a negocio,
2. genera evidencia util para upsell,
3. ayuda a detectar clientes listos para relaciones de mayor ticket,
4. permite validar mercado y readiness,
5. aumenta valor percibido si esta bien acotado.

## Riesgo Economico para la Agencia

Este producto solo sera rentable si no se convierte en performance abierto sin control.

Pierde rentabilidad cuando:

1. se exige optimizacion ilimitada,
2. se abren demasiadas variables a la vez,
3. se vende sin prerrequisitos claros,
4. se intenta suplir con pauta una base comercial defectuosa,
5. se toma responsabilidad por procesos comerciales que siguen siendo del cliente.

## Decision Provisional

Funnel de Captacion se mantiene como producto candidato valido dentro del portafolio inicial de la agencia.

## Pendientes para Cierre Final

Antes de declararlo cerrado como paquete comercial definitivo, aun falta definir:

1. nombre comercial final,
2. numero maximo de campañas o conjuntos activos,
3. limites de optimizacion incluidos,
4. tiempo de activacion y lectura inicial,
5. rango de precio,
6. criterio exacto para distinguirlo de Impulso Inicial de Demanda,
7. si la fase recurrente posterior sera parte de este mismo producto o una extension aparte.
