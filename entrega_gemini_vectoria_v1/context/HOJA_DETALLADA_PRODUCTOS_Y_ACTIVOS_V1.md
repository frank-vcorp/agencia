# Hoja Detallada de Productos y Activos V1

**ID:** ARCH-20260423-31  
**Proyecto:** VectorIA (nombre de trabajo)  
**Fecha:** 2026-04-23  
**Objetivo:** concentrar en una sola hoja los productos de VectorIA y los activos que cada uno puede incluir, para tener claridad comercial sobre que si puede entrar dentro de cada modulo y que solo entra segun diagnostico.

## Como leer esta hoja

Esta hoja no significa que todos los clientes reciben todos los activos.

Significa que cada producto tiene:

1. activos base que normalmente si pueden formar parte del modulo,
2. activos opcionales que entran solo si el diagnostico lo justifica,
3. una logica de seleccion para no inflar alcance sin control.

## 1. Lanzamiento Comercial Inicial

**Para que sirve:**
salir al mercado con una base comercial digital minima, clara y funcional.

**Lectura comercial correcta:**
este paso 1 debe venderse como un paquete fijo de entrada.

No debe venderse como proyecto abierto ni como modulo flexible desde la primera conversacion.

Su funcion comercial es simple:

1. entrar rapido,
2. reducir friccion de venta,
3. dejar una base minima real,
4. conseguir el contacto del cliente,
5. meterlo al ecosistema de VectorIA para despues diagnosticar continuidad.

### Paquete fijo sugerido V1

**Precio fijo sugerido:** 12,000 MXN  
**Modalidad:** pago unico  
**Tiempo de implementacion:** 5 a 10 dias habiles

### Incluye exactamente

1. una landing page o pagina de arranque de una sola seccion comercial,
2. una propuesta de valor principal redactada,
3. un copy base de presentacion comercial,
4. una ruta principal de contacto por WhatsApp o formulario,
5. una configuracion basica de identidad visual funcional aplicada al activo,
6. tres piezas graficas base de lanzamiento o presentacion,
7. un texto base de descripcion comercial para perfil o pagina,
8. checklist final de salida a mercado.

### Puede incluir una sola variable segun el caso

1. alta o configuracion de Google Business Profile si el negocio es local,
2. o alta/configuracion de un solo perfil principal, como Facebook, Instagram o LinkedIn,
3. pero no todos al mismo tiempo dentro del mismo precio fijo.

### No incluye dentro del precio fijo

1. sitio web de multiples secciones,
2. branding profundo o manual de marca,
3. ecommerce,
4. automatizaciones avanzadas,
5. campañas mensuales,
6. community management,
7. multiples perfiles montados al mismo tiempo,
8. dominio, hosting o correos,
9. produccion audiovisual premium,
10. cambios ilimitados.

### Regla operativa

Si el cliente pide algo fuera de esta lista, deja de ser paso 1 fijo y pasa a:

1. complementario opcional,
2. ampliacion con costo separado,
3. o un modulo posterior del portafolio.

**te damos todos los activos posibles.**

## 9. Regla especial para el paso 1

El paso 1 no debe venderse con demasiadas opciones.

Debe operar como oferta de entrada cerrada.

Mientras mas claro, fijo y repetible sea, mas facil sera:

1. venderlo rapido,
2. no improvisar cotizaciones,
3. proteger margen,
4. convertirlo en puerta de entrada comercial al ecosistema.
