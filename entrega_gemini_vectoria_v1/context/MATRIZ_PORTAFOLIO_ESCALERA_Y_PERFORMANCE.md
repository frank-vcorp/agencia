## 1. Propósito del Documento

Este documento existe para ver los 4 productos del portafolio en una sola lectura desde tres perspectivas complementarias:

1. comparacion entre productos,
2. linea de tiempo o escalera de madurez del cliente,
3. lectura de performance, impacto y rentabilidad.

La finalidad no es redefinir los productos, sino entender como trabajan juntos dentro del sistema comercial de la agencia.

## 4. Línea de Tiempo del Cliente

La linea de tiempo no debe leerse como una secuencia obligatoria de compra. Debe leerse como la evolucion logica de madurez del cliente.

### T0. Cliente sin base comercial clara

Necesidad dominante:

1. salir al mercado,
2. verse serio,
3. tener punto de contacto,
4. dejar de improvisar.

Producto natural:

**Lanzamiento Comercial Inicial**

### T1. Cliente ya visible pero todavia poco claro

Necesidad dominante:

1. ordenar presencia,
2. alinear mensaje,
3. mejorar presentacion,
4. hacer mas util lo que ya tiene.

Producto natural:

**Presencia que Vende**

### T2. Cliente con base lista para atraer demanda

Necesidad dominante:

1. generar trafico,
2. captar interes real,
3. medir respuesta,
4. validar readiness comercial.

Producto natural:

**Funnel de Captacion**

### T3. Cliente que ya necesita continuidad, autoridad y soporte comercial

Necesidad dominante:

1. sostener presencia,
2. reforzar confianza,
3. apoyar conversion,
4. mantenerse activo sin caos.

Producto natural:

**Motor de Contenido Funcional**

## 5. Línea de Tiempo Resumida

| Momento del cliente | Señal dominante | Producto mas natural | Objetivo de esa fase |
| :-- | :-- | :-- | :-- |
| T0 | Todavia no tiene base seria | Lanzamiento Comercial Inicial | Salir bien al mercado |
| T1 | Ya existe, pero comunica mal | Presencia que Vende | Ordenar y convertir mejor |
| T2 | Ya esta listo para probar crecimiento | Funnel de Captacion | Generar demanda medible |
| T3 | Ya necesita continuidad y autoridad | Motor de Contenido Funcional | Sostener presencia y confianza |

## 6. Entrada Flexible a la Escalera

Aunque existe una linea de tiempo logica, un cliente puede entrar en cualquier punto de la escalera si ya cumple prerrequisitos.

Ejemplos:

1. una empresa nueva entra en Lanzamiento Comercial Inicial,
2. una empresa antigua pero desordenada entra en Presencia que Vende,
3. una empresa con buena base y necesidad clara de leads entra directo en Funnel de Captacion,
4. una marca con captacion activa pero poca presencia sostenida puede entrar directo en Motor de Contenido Funcional.

La llave de entrada correcta sigue siendo siempre el prediagnostico.

## 7. Lectura de Performance

Aqui, performance no significa solo pauta. Significa cercania al ingreso, capacidad de producir evidencia comercial y valor defendible para el cliente.

| Producto | Cercania al ingreso | Tipo de performance |
| :-- | :-- | :-- |
| Lanzamiento Comercial Inicial | Media | Performance estructural: prepara el terreno para vender |
| Presencia que Vende | Media a alta | Performance de conversion: mejora claridad y confianza |
| Funnel de Captacion | Alta | Performance directa: trae demanda medible |
| Motor de Contenido Funcional | Media | Performance de soporte: sostiene autoridad, confianza y continuidad |

## 7.2 Performance por Velocidad de Evidencia

| Producto | Velocidad para mostrar evidencia | Tipo de evidencia que genera |
| :-- | :-- | :-- |
| Lanzamiento Comercial Inicial | Media | Presencia montada, base funcional, salida a mercado |
| Presencia que Vende | Media | Mejora visible de claridad, CTA y orden comercial |
| Funnel de Captacion | Alta | Trafico, contactos, respuesta inicial del mercado |
| Motor de Contenido Funcional | Media a baja | Consistencia, presencia sostenida, apoyo a confianza |

| Producto | Rentabilidad potencial | Motivo |
| :-- | :-- | :-- |

| Presencia que Vende | Alta | Atiende un problema comun y puede estandarizarse bien |

| Motor de Contenido Funcional | Media a alta | Puede ser recurrente, pero exige mucho control de volumen y limites |

## 9. Lectura Estratégica del Portafolio

Los 4 productos no juegan el mismo papel.

### Productos de entrada

1. Lanzamiento Comercial Inicial
2. Presencia que Vende

Estos sirven para cerrar necesidades urgentes, visibles y faciles de explicar.

### Producto de crecimiento

1. Funnel de Captacion

Este es el producto mas cercano a demanda medible y a una narrativa fuerte de resultado.

### Producto de continuidad y soporte

1. Motor de Contenido Funcional

Este sostiene presencia, autoridad y soporte comercial, y puede extender ticket si se controla bien.

## 10. Lectura Integrada de la Escalera

La forma mas correcta de entender el sistema es esta:

1. Lanzamiento crea base.
2. Presencia ordena y fortalece.
3. Funnel activa crecimiento.
4. Contenido sostiene presencia y refuerza conversion.

Y adicionalmente:

5. Impulso Inicial de Demanda puede aparecer como prueba tactica dentro de Lanzamiento o Presencia si el cliente ya cumple condiciones minimas.

## 11. Qué se Ve en el Tiempo y en Performance

### En el tiempo

Los productos avanzan junto con la madurez del cliente.

### En performance

Los productos se acercan gradualmente al ingreso:

1. primero preparan terreno,
2. luego mejoran conversion,
3. luego generan demanda,
4. luego sostienen continuidad y confianza.

### En rentabilidad

No todos pesan igual:

1. los productos de entrada facilitan cierre,
2. el de captacion eleva valor economico directo,
3. el de contenido sostiene permanencia y soporte de ticket.

## 12. Conclusion Ejecutiva

La cartera ya se puede leer como un sistema coherente.

No son cuatro servicios aislados.

Son cuatro respuestas distintas para momentos distintos del cliente, con distinta cercania al ingreso, distinta complejidad y distinta funcion economica dentro de la agencia.

La regla rectora se mantiene:

**entrada flexible por diagnostico, evolucion natural por madurez, y crecimiento del ticket conforme aumenta el valor del problema resuelto.**
