# Prompt para Gemini - VectorIA V1

Usa los documentos adjuntos para construir una version visual, bonita y consistente de la marca y oferta de VectorIA.

Quiero que trabajes con esta logica:

1. no reinterpretes el modelo desde cero,
2. parte de los documentos como fuente de verdad,
3. detecta contradicciones menores y favorece la capa comercial mas reciente,
4. transforma la informacion en piezas visuales, claras y profesionales.

Los entregables que quiero que propongas o desarrolles son estos:

1. una presentacion bonita de la agencia,
2. una hoja comercial de una sola pagina,
3. una tabla visual del portafolio con precios y tiempos,
4. una propuesta visual para socios,
5. una estructura de sitio web o landing para VectorIA.

Condiciones importantes:

1. el nombre de trabajo es VectorIA,
2. el producto 1 ya es paquete fijo de entrada por 12,000 MXN,
3. los productos 2, 3 y 4 son modulos recurrentes,
4. Impulso Inicial de Demanda es validacion tactica,
5. dominio, hosting y correos son complementarios basicos opcionales,
6. el tono debe ser sobrio, comercial, moderno y claro,
7. evitar estetica de software frio, cripto, robotica o agencia generica de redes,
8. priorizar una comunicacion que se sienta orientada a sistema y resultado.

Si propones una web, su estructura minima deberia incluir:

1. hero claro,
2. explicacion del problema que resuelve la agencia,
3. portafolio modular,
4. producto de entrada destacado,
5. tabla simple de precios o lectura comercial,
6. complementarios basicos,
7. llamada a la accion final.

Usa los documentos del paquete como respaldo y no inventes productos nuevos fuera de lo que ya esta definido.
